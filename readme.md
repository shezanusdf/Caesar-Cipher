# Caesar-Cipher

A simple terminal based caesar cipher endcoder/decoder program made in python.

## Features

- Press E for (E)ncryption
- Press D for (D)ecryption
- Ctrl + C to quit the program

## How to run

1. Clone this repo:
```bash
git clone https://github.com/shezanusdf/caesar-cipher
```

2. Run the program
```bash
python caesar-cipher.py
```

3. (E)ncrypt or (D)ecrypt:
```bash
Do you want to (e)ncrypt or (d)ecrypt?
> 
```

4. Enter your message:
```bash
Enter message to (e)ncrypt or (d)ecrypt?
>
```

5. Enter Key:
```bash
Enter Key: 21
```
